Космос Тертеряна

16 и 17 июня в 19:00 в Креативном кластере «Л52» Центр авангарда проводит музыкальный фестиваль «Космос Тертеряна»

Авет Тертерян - авангардный композитор мирового уровня. Его музыка самобытная, свободная, характерная для 20 века, каждый звук - яркий образ. Она вбирает в себя, с одной стороны, национальную традицию, а, с другой - рефлексию композитора на актуальные события, через которые прошла Армения в 20 веке.


На фестивале прозвучит музыка Авета Тертеряна, Ольги Викторовой, Сергея Патраманского, Дмитрия Ремезова.

Программа:
  • Струнный квартет № 1, До-мажор в двух частях.
  • Сергей Патраманский. «Лики звёзд» для камерного ансамбля.
  • Дмитрий Ремезов. «deep bunker M.» для камерного ансамбля и электроники.
  • Сергей Патраманский. «Невесомое» для камерного ансамбля.
